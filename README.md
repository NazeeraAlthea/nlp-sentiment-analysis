# NLP Sentiment Analysis

This project performs **sentiment analysis** using Natural Language Processing (NLP) to classify text as positive, negative, or neutral.  
Sentiment analysis helps computers understand human emotion in text, which is widely used in applications like review analysis, social media monitoring, and customer feedback analysis. :contentReference[oaicite:1]{index=1}

This repository includes:
